```javascript
strings.sort((a, b) => {});
```

위의 코드에서 a, b는 next, prev와 같다. 즉, sort((next, prev)=>{});이다.


반환 값이 0보다 크다면 a 먼저, 반환 값이 0이라면 그대로, 반환 값이 0보다 작다면 b가 먼저 온다.
